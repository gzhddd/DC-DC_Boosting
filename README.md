# DC-DC_Buck
## MP2315S 降压型 DC-DC 电源模块

### 1. 模块简介
* 本电路是一个基于 MP2315S 同步降压转换器芯片设计的高效直流电源转换模块。
* 其主要功能是将 +12V 的输入主供电进行稳定降压，输出至后级负载（图纸标注为 +SV，结合典型应用场景通常指代 +5V 输出）。

### 2. 核心元件配置
* **主控芯片**：采用 MP2315S 作为核心的开关稳压 IC。
* **输入保护**：输入端配备了 SMAJ12A 瞬态抑制二极管（TVS），主要用于吸收电路中的浪涌电压，从而保护后级核心器件免受过压损坏。
* **储能与滤波**：开关节点后级连接了主功率电感（约 4.7uH）。输出端则配备了 22uF 的大容量滤波电容，用于大幅降低输出纹波并维持电压稳定。

### 3. 关键引脚与网络说明

| 引脚/网络名称 | 功能描述 |
| :--- | :--- |
| **IN** | 电源输入端，直接连接 +12V 主供电。该节点旁路配置了 22uF 和 0.1uF 的输入滤波电容，以滤除高频噪声。 |
| **SW** | 开关输出节点（Switch），直接连接至功率电感。 |
| **BST** | 自举引脚（Bootstrap），通过外部电容（如 0.1uF）为高侧内部 MOSFET 提供驱动电压。 |
| **EN** | 使能控制引脚（Enable），外围通过 300K 和 100K 的分压电阻网络进行电平配置。 |
| **FB** | 电压反馈引脚（Feedback），连接 40.2K 和 20K 的反馈分压电阻网络以设定最终的输出电压，同时并联了 33pF 的前馈电容以提升系统的动态响应和稳定性。 |
| **VCC** | 内部 LDO 输出，通常需外接 0.1uF 旁路电容以保证内部逻辑电路稳定运行。 |
| **AAM** | 高级异步调制引脚（Advanced Asynchronous Mode），外接 1.09K 电阻用于设置轻载高效模式或强制连续导通模式的阈值。 |
| **GND** | 模块公共接地端。 |
